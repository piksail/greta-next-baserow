# TP Next.js / React

Ce projet regroupe différents exercices pour mettre en pratique ce que nous avons vu à propos de Next.js, React et autres outils pour vous aider dans le développement web. Pour chaque exercice, vous pouvez créer un repo dédié ou utiliser le système de pages de Next.js.
